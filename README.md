## Installation
To install AiFlow, follow the steps below:

1. Build the package:

```bash
python setup.py sdist bdist_wheel
```

1. Install the package from the generated wheel:

```bash
pip install dist/aiflow-2.0.0-py3-none-any.whl --force-reinstall
```
